Replication files for the paper "Precautionary savings in small open economy"
Liudmila Kiseleva, 2021

The directory consists the following files:

1) run.m runs the whole code

2) Calibration.mod and Parameters.m declare parameters values from CMRR

3) Smoother.mod and Declaration.mod structure the model in different settings 

4) Model.mod describes the model in different settings

5) computeMeans.m and estimationData.mat are files from CMRR necessary to solve the model
